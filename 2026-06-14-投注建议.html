大小球条件概率深度分析: 零封比分的边际成本优化

核心原理

零封不是单一比分，而是一个比分集合。每多选一个比分 = 成本+2元。用泊松条件概率（给定弱队0球） × 大小球预期总进球 精确锁定最优覆盖边界。

结论: 2选 (0:1 + 0:2) 是最优解

  1. 大小球预期总进球 2.46 → 最可能的零封比分是 0:2 和 0:3 → 与总进球2~3球区间一致
  2. 0:1 (11.7%) + 0:2 (9.6%) = 覆盖 56.2% 零封场景 → 2注4元 → 最低回报10.4元 → 2.60x回报比
  3. 3选加0:3: 覆盖率升至69.9% (+13.7%)，成本+2元 → 边际增益适中，可选但不必须
  4. 不推荐加0:0: 0:0是平局非瑞士胜，与预测方向不一致 → 除非做对冲，否则不入零封比分复选
  5. 0:4以上: 累计覆盖仅11%，每+1比分成本+2元但覆盖率增量递减 → 边际递减，不划算
  6. 核心口诀: λ_total ≈ 大小球盘口 → 总进球±0.5的零封比分 = 最优区间
