🔴 复式串关铁律（v2.0→v3.0 核心修正）

❌ 跨玩法混选 = 违规

同一场比赛不能同时选多个玩法：
半全场-胜胜 + 比分-2:0 → 违规
胜负平-主胜 + 总进球-2球 → 违规
选了半全场就不能再选比分/胜负平/总进球

✅ 同一玩法内多选 = 合规

同一场比赛在同一玩法内可选多个结果：
总进球-1球 + 总进球-2球 → 合规
比分-2:0 + 比分-1:0 + 比分-3:0 → 合规
半全场-胜胜 + 半全场-平胜 → 合规

📌 一句话口诀：同一场比赛，选了一个玩法就死磕到底，在该玩法内多选结果算复式，跨玩法混选直接废。

📚 竞彩串关规则详解（v3.0增强版）

一、什么是"复式"？

  • 复式 = 同一场比赛 + 同一玩法 + 多个结果
  • 例如：比分玩法选 2:0 + 1:0 + 3:0 → 系统自动覆盖3种比分的所有串关组合
  • 例如：总进球玩法选 1球 + 2球 → 系统自动覆盖2种总进球的组合
  • 不能跨玩法混选：同一场比赛选了"比分"就不能再选"半全场/总进球/胜负平"

二、什么情况下必须用不同玩法？

  • 不同比赛之间可以自由混选玩法（推荐）：A场选比分、B场选总进球、C场选胜负平 → 完全合规
  • 这样做的优势：各场比赛选自己"性价比最高"的玩法，不受同一玩法的限制
  • 例如方案B中：墨选总进球 + 韩选胜负平 + 加选总进球，三场用了两种玩法，合规且最优
